# Asteroids

My A Level project to recreate Asteroids using C# alongside MonoGame. It has other features such as a powerup system.

## Built With

* [Visual Studio](https://visualstudio.microsoft.com/) - IDE Used
* [MonoGame](http://www.monogame.net/) - Game framework used
